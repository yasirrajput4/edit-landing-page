# Landing Page & Content Editor

A lightweight, customizable landing page with a built‑in content editor. The editor lets you update the main hero section of the page and preview changes instantly, with support for dark mode, responsive design, and local persistence via `localStorage`.

## Features

- **Customizable content**: Edit headline, subheadline, and description.
- **Instant updates**: See changes applied immediately to the landing page.
- **Responsive layout**: Optimized for desktop, tablet, and mobile devices.
- **Dark mode**: Toggleable dark theme for comfortable viewing.
- **Local persistence**: Saves content in the browser using `localStorage`.
- **Lightweight**: Pure HTML, CSS, and vanilla JS with zero external dependencies.

## Tech Stack

- **HTML5** for structure
- **CSS3** for layout, theming, and dark mode
- **Vanilla JavaScript** for content editing, state management, and persistence

## Project Structure

- `html/index.html` – Public landing page.
- `html/editor.html` – Content editor interface.
- `css/index.css` – Styles for the landing page.
- `css/editor.css` – Styles for the editor view.
- `js/index.js` – Logic for rendering the landing page and theme toggling.
- `js/editor.js` – Logic for the content editor and data synchronization.

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yasirrajput4/edit-landing-page.git
   cd edit-landing-page
   ```
2. **Run the App**

- Open html/index.html directly in your browser.

3. **Open the Editor**

- Click the "Edit Landing Page" button on the main page to customize your site.
